# NetSpy

A local network analyzer and security dashboard for auditing your own networks.
Key Features

    📊 Traffic Monitoring: Real-time charts for incoming and outgoing traffic.

    🛡 Security Score: Heuristic analysis of system security.

    📍 Geolocation: Visualization of your external IP on a map.

    ⚡ Auto-update: The program automatically checks for new versions.

How to run

    Download the latest .exe version from the Releases section.

    Run the file. The dashboard will open in your browser automatically.
